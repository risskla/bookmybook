>>>>>>>>> Match user <<<<<<<<<<
Stocker dans une table : 
User, livre, Ecart

pour calculer l'écart des notes globale entre le user tous les autres users sur les livres qu'il a lu

calculer ensuite : 
User, MoyenneEcart 

Pour le lecteur le plus proche : prendre le user correspondant au min de MoyenneEcart; en cas d'égalité on prend le premier ; 
Pour le lecteur le moins proche : prendre le user correspondant au max de MoyenneEcart; en cas d'égalité on prend le premier ;
